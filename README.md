
# Eddie's Blog

这是我的个人博客，基于 [VitePress](https://vitepress.dev/) 构建。

在这里，我记录了我的学习笔记、项目经验和一些个人思考。

## 运行

1. **安装依赖**

   ```bash
   pnpm install
   ```

2. **本地开发**

   ```bash
   pnpm docs:dev
   ```

3. **构建**

   ```bash
   pnpm docs:build
   ```

## 内容

本博客主要包含以下几个部分：

-   **学习笔记**: 涵盖前端工程化、计算机网络、CSS、JavaScript、React、Vue、Vite 等领域的学习记录。
-   **面试整理**: 整理了一些面试真题和准备过程中的思考。
-   **项目实践**: 记录了一些个人项目和有趣功能的实现。
-   **随笔**: 一些关于技术、学习和生活的随想。
